## ✨ Features

- 🚀 스크롤 카드 자동 스냅 – 스크롤 위치에 따라 자연스럽게 카드가 정렬됩니다.
- 🎨 GSAP – hook을 사용하여 다양한 애니메이션 생성
- 📌 프로젝트 상세 페이지 – 참여한 프로젝트 및 기술 설명 제공
- 📱 반응형 디자인 – 모바일, 태블릿, 데스크톱 모두 최적화
